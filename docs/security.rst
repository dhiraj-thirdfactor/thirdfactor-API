Security & Privacy
==================

HTTPS Requirements
------------------

All traffic to the ThirdFactor AI API Gateway must be encrypted via **HTTPS** (TLS 1.2 or higher). Unencrypted HTTP requests will be rejected to ensure data integrity and confidentiality.

Compliance
----------

ThirdFactor AI is committed to protecting user privacy and ensuring compliance with global and local data protection regulations.

GDPR (General Data Protection Regulation)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

We adhere to GDPR principles regarding the processing of personal data for individuals within the European Union. This includes:

*   **Data Minimization**: Only necessary data is processed.
*   **Encryption**: Personal data is encrypted in transit and at rest.
*   **Right to Access/Erasure**: Mechanisms are in place to support user requests for data access or deletion.

NDPA (Nigeria Data Protection Act)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

For operations involving data subjects in Nigeria, we comply with the NDPA framework, ensuring:

*   Lawful processing of personal data.
*   Protection of data subject rights.
*   Implementation of adequate technical and organizational security measures.

*Note: Please refer to our full Privacy Policy for detailed information on data handling practices.*
